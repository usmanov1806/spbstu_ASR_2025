**Важно!**<br>
В качестве отчетности по проделанной работе просьба отправлять в общий чат с преподавателем **ссылки на гугл/яндекс-документы** с материалами (презентации, доп.материалы). Папка на диске после каждого выступления будет дополняться текстовым документом с дублированием всех ссылок из вашего сообщения. Пример файла .txt есть в папке на диске по первой лабораторной работе. Просьба присылать именно в таком формате, т.к. объем личного яндекс-диска ограничен.


## Правила загрузки лабораторных работ в репозиторий

<ol>
    <li>
       Лабораторные работы необходимо загружать в свою папку (необходимо будет создать личную папку, название = ваша фамилия)
    </li>
    <li>
       Название файла = номер лабораторной работы по списку
    </li>
    <li>
        Для каждой лабораторной работы назначен дедлайн, который оглашается на паре
    </li>
    <li>
        При загрузке лабораторных необходимо будет форкнуть репозиторий и создавать PR в ветку main, чтобы ваши изменения отобразились в общем репозитории
    </li>
</ol>
